# C Variables, if else and while

- This directory consists of the illustration of variables, if, else, while loops in C language.
- It consists of C programs, Shell scripting, C source files and C executables.

# Tasks: 

0. <b> Positive anything is better than negative nothing </b>- program will assign a random number to the variable n each time it is executed. Complete the source code in order to print whether the number stored in the variable n is positive or negative.
1. <b> The last digit </b> -  program will assign a random number to the variable n each time it is executed. Complete the source code in order to print the last digit of the number stored in the variable n.
2. <b>I sometimes suffer from insomnia. And when I can't fall asleep, I play what I call the alphabet game </b> -program that prints the alphabet in lowercase, followed by a new line.
3. <b>alphABET </b>- a program that prints the alphabet in lowercase, and then in uppercase, followed by a new line.
4. <b>When I was having that alphabet soup, I never thought that it would pay off </b>-  a program that prints the alphabet in lowercase, followed by a new line.
5. <b>Numbers </b>- a program that prints all single digit numbers of base 10 starting from 0, followed by a new line.
6. <b>Numberz </b>- a program that prints all single digit numbers of base 10 starting from 0, followed by a new line
7. <b>Smile in the mirror </b> - a program that prints the lowercase alphabet in reverse, followed by a new line.
8. <b>Hexadecimal </b>- a program that prints all the numbers of base 16 in lowercase, followed by a new line.
9. <b>Patience, persistence and perspiration make an unbeatable combination for success </b>- a program that prints all possible combinations of single-digit numbers, separated by a comma and space, in ascending order.
10. <b>Inventing is a combination of brains and materials. The more brains you use, the less material you need </b>- a program that prints all possible different combinations of two digits, separated by a comma and space, the two digits must be different.
11. <b>The success combination in business is: Do what you do better... and: do more of what you do...</b> -  a program that prints all possible different combinations of three digits, separated by a comma and space, the three digits must be different.
12. <b>Software is eating the World </b>- a program that prints all possible combinations of two two-digit numbers.
